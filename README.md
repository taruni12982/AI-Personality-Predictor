# AI Personality Predictor from Writing

This project predicts MBTI personality type from written text using Natural Language Processing and Machine Learning.

## Features
- MBTI personality prediction
- TF-IDF text vectorization
- Naive Bayes classification
- Personality interpretation
- Probability visualization graphs
- Interactive Streamlit dashboard

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib

## Dataset
MBTI Personality Dataset from Kaggle

## Machine Learning Workflow
1. Text preprocessing
2. TF-IDF feature extraction
3. Model training using Multinomial Naive Bayes
4. Personality prediction
5. Probability visualization

## Run the Project

```bash
pip install streamlit pandas matplotlib scikit-learn
streamlit run app.py
```

## Project Output
The application predicts:
- Personality Type
- Energy Style
- Decision Style
- Lifestyle Pattern
